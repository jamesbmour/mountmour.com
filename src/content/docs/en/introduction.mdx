## Bmour Welcome Letter

---

**Welcome to Astro 2.0!**

This is the `docs` starter template.
It contains all of the features that you need to build a Markdown-powered documentation site, including:

- 📝 **Full Markdown and MDX support**
- 📱 **Responsive mobile-friendly design**
- 🧭 **Sidebar navigation**
- 🔎 **Search (powered by Algolia)**
- 🔤 **Self-Hosted webfont (GDPR compliant)**
- 🗨️ **Multi-language i18n**
- 📃 **Automatic table of contents**
- 🧑‍💻 **Automatic list of contributors**
- 🌙 (and, best of all) **dark mode**
